# Cave Story Remake

This is a remake of a popular platform game, [Cave Story](https://en.wikipedia.org/wiki/Cave_Story), using original assets. The libraries used were SDL2 and tinyXML. 

## Features

The gameplay includes:

- Different Levels
- Collisions
- Jumping
- Enemies
- Sprite animations
- Health
- Heads-up display
